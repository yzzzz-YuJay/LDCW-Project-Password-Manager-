# README - Smart Password Security Checker

===================================

**Smart Password Security Checker** is a C++ console application developed for the LDCW6123 Fundamentals of Digital Competency group project.

This program is inspired by **Cybersecurity - Password Manager** technology. It evaluates the strength of a user's password based on several security criteria and provides suggestions to improve password security.

---

## Main Features

* Check password strength
* Calculate password security score (0 - 100)
* Classify password as **Weak**, **Medium**, or **Strong**
* Display password improvement suggestions
* View password security tips
* View program information
* User-friendly menu interface

---

## How to Use

### Main Menu

When the program starts, the following menu will be displayed:

```
[1] Check Password
[2] Password Security Tips
[3] About Program
[4] Exit Program
```

### Option 1 - Check Password

* Select **1** from the main menu.
* Enter any password you would like to evaluate.
* The system will analyze your password and display:

  * Password security score
  * Password strength (Weak / Medium / Strong)
  * Password improvement suggestions

* After the analysis, choose:

  * **1** to check another password
  * **2** to return to the main menu

### Option 2 - Password Security Tips

Displays several recommended practices for creating strong and secure passwords.

### Option 3 - About Program

Displays the program information and project details.

### Option 4 - Exit Program

Closes the application.

---

## Password Evaluation Criteria

The program checks whether the password contains:

* At least 8 characters
* Uppercase letters
* Lowercase letters
* Numbers
* Special characters

A security score is calculated based on these criteria.
